Projects

UnMute

Unmute is a web application that uses Symphonic Labs’ API to analyze non-verbal lip movements and generate real-time speech and captions from video input, aimed at enhancing accessibility. The app was developed using Agile principles, with iterative testing involving over 20 deaf and hard-of-hearing users to ensure it effectively enabled communication via lip-reading before final deployment.

ImageWizard

ImageWizard is a full-stack AI-powered image generator built using React, Node.js, and MongoDB. It uses OpenAI’s API to generate images based on user prompts and integrates Cloudinary for media storage, achieving a 40% reduction in load times. The application is deployed with Netlify for the frontend and Render for the backend, providing a scalable and efficient asynchronous workflow that ensures seamless real-time image rendering for users.

Zombie Walking Deadline

Zombie Walking Deadline is a custom-built clone of the popular game Plants vs. Zombies, developed using multiple algorithms including stacks, deques, and queues. The implementation maintained linear space and time complexity through the strategic use of hash tables to optimize performance and resource management..
